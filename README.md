# esp32_picow_robot
This project allows a Pico W to act as a controller for an ESP32 robot.

## Getting Started

Make sure to have the latest version of MicroPython on the Pico and ESP32. Also the ESP32 doesn't have its own version of `aioble` so be sure to download the `micropython-libs` directory from GitHub and then copy the contents to a directory which should be called libs within Thonny. Be sure to create the directory first. Other than that rename both files on each side `main.py` and then run them headlessly. Otherwise that's it.
